# Dankoe Skill Source Map

This public file explains how the skill toolbox relates to source materials.

The repository does not include Dan Koe's original courses, books, newsletters, YouTube transcripts, paid materials, or the author's private local knowledge base.

## What Users Can Read

Users can read and use the bundled abstract knowledge packs:

```text
知识库/Skill知识包/
├── dankoe_学习地图.md
├── dankoe_IMA桥接.md
├── dankoe_一人公司框架.md
├── dankoe_offer框架.md
├── dankoe_内容地图.md
├── dankoe_写作系统.md
└── dankoe_AI工作流.md
```

These files are portable workflow notes. They are included in the relevant skill zip packages.

## What Users Cannot Read

Users cannot read the author's private local source folder unless they also have those files on their own machine and provide the path to their agent.

Any path like this is local-only and not part of the public package:

```text
<your-local-dankoe-source-folder>
```

## How To Use With Your Own Source Library

If you have your own local Dankoe source library, tell the agent:

```text
我的 Dankoe 资料库在：{your path}
请结合 $dankoe-learning-map 给我做学习路线。
```

The skills will still work without a source library, but they will use only the bundled abstract workflow notes rather than full-source retrieval.
